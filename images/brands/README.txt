ALLURE — brand logos for the "Brands we carry" strip on the homepage
====================================================================

The homepage looks for a logo image for each brand IN THIS FOLDER. If a file is
missing it tries an online logo service, and if that fails it shows the brand
name as text — so the strip never looks broken.

To show real logos, save each brand's logo here using EXACTLY these file names
(lowercase, .png, ideally transparent background, ~200px wide):

    chanel.png
    dior.png
    gucci.png
    ysl.png
    versace.png
    creed.png
    bvlgari.png
    hugo-boss.png
    armani.png
    carolina-herrera.png
    paco-rabanne.png
    tom-ford.png
    parfums-de-marly.png
    amouage.png
    nishane.png
    valentino.png

Tips:
- PNG with a transparent background looks best (the strip greyscales logos and
  shows full colour on hover).
- .jpg works too, but then change the file extension in index.html to match,
  or just convert to .png.
- Only add logos for brands you actually sell. To change the brand list, edit
  the "brands-band" section in index.html.

Note: brand logos are trademarks of their owners. Use the official logos only
to represent products you genuinely stock (standard retailer practice).
